\# PID Temperature Control System



\## 🔧 Overview

A closed-loop PID-based temperature control system implemented using Arduino and DHT sensor.



\## ⚙️ Features

\- Real-time temperature sensing

\- PID control (Proportional–Integral–Derivative)

\- PWM-based actuator control

\- Serial monitoring for debugging



\## 🧠 Working

The system continuously measures temperature and compares it with a setpoint.  

The PID controller adjusts output to minimize error and stabilize temperature.



\## 🧰 Components

\- Arduino Uno

\- DHT11 Sensor

\- Resistors, LED / Heater



\## 📌 Applications

\- Industrial temperature control

\- Incubators

\- HVAC systems
